แพ็กเกจ deploy โค้ดอย่างเดียว (ไม่มีเนื้อหาลูกค้า)

ไม่ได้ใส่ในแพ็กเกจ (โฮสต้องเก็บของเดิม):
  * data/site.json (ข้อความ ราคา โปรแกรม ทั้งหมด)
  * data/backups/
  * assets/uploads/ (รูป)
  * assets/js/data-fallback.js

วิธีอัป:
  1. แบ็คอัพบนโฮสก่อน (admin > สำรอง & กู้คืน)
  2. อัปโหลด merge ขึ้นโฮส (อย่าลบ data/ บนโฮส)
  3. รีเฟรชหน้าเว็บ + ทดสอบบันทึก admin

ถ้าข้อมูลหาย: กู้คืนจาก admin > สำรอง & กู้คืน
